## Additional SEO Signals

### Programmatic Page Templates to Build

1. **Language detail page** — `/languages/{slug}` — canonical facts, relationships, timeline position
2. **"Written in" page** — `/written-in/{language}` — e.g. `/written-in/c` lists all languages whose runtime is C
3. **Lineage page** — `/lineage/{language}` — ancestry tree from a single language
4. **Comparison page** — `/compare/{lang-a}/{lang-b}` — side-by-side timeline + relationship data
5. **Tool/Runtime page** — `/tools/{slug}` — e.g. `/tools/llvm`, `/tools/beam-vm`
6. **Era/decade page** — `/timeline/{decade}` — all languages released in a given decade
