About J

J is a programming language. It is a dynamically typed language. It supports array and functional programming.

J first appeared in 1990 and was designed by Kenneth E. Iverson. Development is led by Roger Hui.

How J is implemented

In the Language Lineage dataset, its runtime is implemented in C.
