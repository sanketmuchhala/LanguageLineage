About Bourne Shell

Bourne Shell is a command line interpreter for operating systems. It was first released in 1979. Development is led by Stephen R. Bourne.

In the Language Lineage dataset, its runtime is implemented in C.

Bourne Shell drew on ideas from ALGOL 68 and went on to influence Bash and Perl.
