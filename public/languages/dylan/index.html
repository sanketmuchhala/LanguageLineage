About Dylan

Dylan is a multi-paradigm programming language with support for functional and object-oriented programming. It is a dynamically typed language. It supports object-oriented and functional programming.

Dylan first appeared in 1992. Development is led by Apple Inc. and Carnegie Mellon University.

How Dylan is implemented

In the Language Lineage dataset, its runtime is implemented in C.

Dylan in the language family tree

Dylan drew on ideas from Lisp, Common Lisp Object System, and Common Lisp and went on to influence Python.
