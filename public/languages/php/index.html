About PHP

PHP is a scripting language focused on web development. It was first released in 1995 and was designed by Rasmus Lerdorf. Development is led by Andi Gutmans and PHP Group.

In the Language Lineage dataset, its runtime is implemented in C.

PHP drew on ideas from Perl, C, and Java.
