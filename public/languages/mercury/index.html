About Mercury

Mercury is a functional logic programming language. It is a statically typed language. It supports logic and functional programming.

Mercury first appeared in 1995. Development is led by University of Melbourne.

How Mercury is implemented

In the Language Lineage dataset, its toolchain was bootstrapped from Prolog.
