About Agda

Agda is a dependently typed, purely functional programming language and proof assistant. It is a statically typed language. It supports functional and proof-assistant programming.

Agda first appeared in 2007. Development is led by Chalmers University of Technology.

How Agda is implemented

In the Language Lineage dataset, its compiler is written in Haskell.
