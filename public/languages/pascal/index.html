About Pascal

Pascal is a programming language. It was first released in 1970 and was designed by Niklaus Wirth.

In the Language Lineage dataset, Pascal is self-hosting, so its own compiler is written in Pascal itself.

Pascal drew on ideas from ALGOL and went on to influence Ada, D, Go, Modula-2, Oberon, and Delphi.

  • Ada — Ada designed as Pascal successor for DoD
  • Oberon — Oberon designed by Pascal's creator Wirth as successor
  • Nim — Nim influenced by Pascal's syntax
  • Go — Pascal influenced Go's declaration syntax
  • D — Pascal influenced D's module system
