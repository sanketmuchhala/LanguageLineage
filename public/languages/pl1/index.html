About PL/I

PL/I is a procedural, imperative computer programming language. It is a statically typed language. It supports imperative and procedural programming.

PL/I first appeared in 1964 and was designed by IBM.

How PL/I is implemented

In the Language Lineage dataset, its compiler is written in Assembly.

PL/I in the language family tree

PL/I drew on ideas from COBOL and Fortran.
