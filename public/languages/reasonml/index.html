About ReasonML

ReasonML is a syntax extension and toolchain for OCaml. It is a statically typed language that compiles ahead of time to native machine code. It supports functional and object-oriented programming.

ReasonML first appeared in 2016. Development is led by Meta.

How ReasonML is implemented

ReasonML is primarily implemented in OCaml.
