About Simula

Simula is a programming language. It is a statically typed language that compiles ahead of time to native machine code. It supports object-oriented and imperative programming.

Simula first appeared in 1962 and was designed by Ole-Johan Dahl. Development is led by Kristen Nygaard.

How Simula is implemented

In the Language Lineage dataset, its compiler is written in ALGOL.

Simula in the language family tree

Simula drew on ideas from ALGOL and went on to influence C++, Smalltalk, Java, and Eiffel.
