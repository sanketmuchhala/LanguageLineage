About S

S is a statistical programming language. It is a dynamically typed language. It supports statistical and functional programming.

S first appeared in 1976. Development is led by Richard Becker and John Chambers.

How S is implemented

In the Language Lineage dataset, its runtime is implemented in C and Fortran.

S in the language family tree

S drew on ideas from C and went on to influence R.
