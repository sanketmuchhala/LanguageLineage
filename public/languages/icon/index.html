About Icon

Icon is a programming language. It is a dynamically typed language. It supports imperative and functional programming.

Icon first appeared in 1977 and was designed by Ralph Griswold.

How Icon is implemented

In the Language Lineage dataset, its compiler is written in C.

Icon in the language family tree

Icon drew on ideas from SNOBOL and ALGOL and went on to influence Python.
