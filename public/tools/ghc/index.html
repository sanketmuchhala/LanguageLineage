About GHC

GHC is a compiler for the Haskell programming language. It was first released in 1992. Development is led by University of Glasgow.

In the Language Lineage dataset, GHC is self-hosting, so its own compiler is written in GHC itself, its runtime is implemented in C, and its toolchain was bootstrapped from Lazy ML.
