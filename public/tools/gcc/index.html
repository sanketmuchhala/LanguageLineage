About GCC

GCC is an optimizing compiler produced by the GNU Project, key component of the GNU tool-chain and standard compiler for most projects related to GNU and the Linux kernel. It was first released in 1987. Development is led by GNU Project.

In the Language Lineage dataset, GCC is self-hosting, so its own compiler is written in GCC itself.
