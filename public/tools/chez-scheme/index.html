About Chez Scheme

Chez Scheme is a R6RS Scheme compiler and run-time.

Chez Scheme first appeared in 1985. Development is led by Cisco and Cadence Research Systems.

How Chez Scheme is implemented

In the Language Lineage dataset, Chez Scheme is self-hosting, so its own compiler is written in Chez Scheme itself. Reaching self-hosting — where a language is mature enough to compile itself — is a milestone that proves the language can handle a large, real-world program.
