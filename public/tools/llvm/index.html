About LLVM

LLVM is a compiler backend for multiple programming languages. It was first released in 2003 and was created by Vikram Adve and Chris Lattner. Development is led by Craig Topper and Simon Pilgrim.

In the Language Lineage dataset, its compiler is written in C++.
